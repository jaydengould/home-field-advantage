# home-field-advantage

A cross-sport causal inference study estimating the causal effect of crowd
presence on home-team win probability. It uses the 2020 empty-stadium COVID
season as a natural experiment across MLB, NBA, and NFL. Each sport is analyzed
separately, then combined into a cross-sport comparison, with a paper-quality
write-up rendered via Quarto (PDF + HTML).
